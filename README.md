# wilddog sdk实现屏幕共享
使用wilddogvideo-room sdk实现屏幕共享的demo
## 使用方式：
1. 使用node执行https_channel_server.js：

	node https_channel_server.js

2. 在浏览器中输入：https://localhost:8080来访问，如果8080端口被占用了，可以通过修改https_channel_server.js文件来更改demo使用的端口。
